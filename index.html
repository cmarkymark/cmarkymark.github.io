Tweeting Under the Influence

It has been hypothesized that alcohol-related twitter data can be utilized as a proxy measure for actual acohol consumption. This project was designed to test this hypothesis by collecting and analyzing alcohol-related twitter data from San Diego, CA.

What To Do?

With new technologies have come new possibilities for leveraging big data to understand human behavior.

If there is a way to utilize social media data to identify and predict alcohol misuse, we can empower the public health sector to better address today's alcohol misuse crisis.

Our goal is to determine if twitter data can help predict alcohol consumption behaviors.
